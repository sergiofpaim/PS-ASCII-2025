Experiência e Projetos

NBA - Plataforma de Monitoramento e Registro de Estatísticas em Tempo Real

Desenvolvi uma plataforma multifacetada para monitoramento e registro de estatísticas de jogos da NBA em tempo real. O projeto abrange diferentes versões, incluindo uma aplicação CLI com operações em banco de dados SQL, uma API RESTful e uma interface gráfica utilizando Next.js, integradas a bancos de dados NoSQL como CosmosDB e MongoDB. Além disso, criei dois repositórios, um com Java e outro com C#.

Tecnologias Utilizadas:

Back-end: C# (.NET 8), Java

Front-end: TypeScript, Next.js, React, Redux

Banco de Dados: SQL Server, CosmosDB, MongoDB

Ferramentas e Bibliotecas: Entity Framework Core, Spectre.Console.Cli, Azure Cosmos SDK

Desafios e Aprendizados:
  • Implementação de operações avançadas em SQL Server e adaptação para CosmosDB e MongoDB.
  • Desenvolvimento de interfaces CLI interativas com Spectre.Console.
  • Criação de APIs RESTful e integração com Next.js e React.
  • Gerenciamento de estados complexos com Redux.
  • Exploração do uso de Java para APIs e interfaces gráficas.

Repositório: GitHub - sergiofpaim/NBA

2D Car Game - Jogo de Corrida em 2D

Desenvolvi um jogo de corrida 2D utilizando C# e Windows Forms. Este projeto foi criado para me ajudar a entender como decompor mecânicas reais de carros em um jogo. O jogo apresenta uma simulação básica de corrida, permitindo ao jogador controlar um carro em uma pista 2D.

Tecnologias Utilizadas:

Linguagem de Programação: C#

Framework: Windows Forms

Desafios e Aprendizados:
  • Compreensão e implementação de mecânicas de física relacionadas a veículos.
  • Desenvolvimento de uma estrutura de código organizada para projetos de maior escala.
  • Aprendizado sobre a criação de interfaces gráficas e manipulação de eventos no Windows Forms.
